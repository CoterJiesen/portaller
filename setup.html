How to setup

That is two ways of how to use Portaller. If you want to unblock content only for one device (e.g. iPhone or SmartTV) — find that device in the list at left sidebar and set it up as described.

But if you want to use it on many devices inside your home network at the same time (e.g. listen Pandora on iPhone and watch Netflix on TV) — best way is write Portaller DNS address to the settings of your router. It will push right DNS settings to any devices connected to it.

Android not supported

Due some limitations in Android TLS implementation, Portaller not support any of Android devices at this time ;(

Sorry about that.


Windows 7

  • Click to the "Network" icon at status bar and open "Network center";
  • Select current active connection;
  • Click "Options" at the window appeared;
  • Select "IPv4" row and choose "Options";
  • Put "107.170.15.247" as primary DNS and "77.88.8.8" as secondary in the "Preferred DNS" section.


    • Mac OS X

  • Open "Network" section;
  • Select active connection and click "Options";
  • Open "DNS" section;
  • Click "Add" and put "107.170.15.247" as primary DNS and "77.88.8.8" as secondary.


    • PlayStation 4

  • Navigate to the settings menu ​and select "Network";
  • Select "Set Up Internet Connection";
  • On the next screen select "Custom​";
  • Chose your connection method — "Wi-Fi" or "LAN Cable";
  • Set obtain your IP address automatically;
  • DHCP host name must not be specified — select "Do Not Specify";
  • Next is your DNS settings — select "Manual";
  • Put "107.170.15.247" as primary DNS and "77.88.8.8" as secondary;
  • Once you have done this press next and then test your connection.

    • XBOX One

  • Go to the Dashboard and press the "Options" button on Xbox One controller;
  • Select "Settings" from the menu that will appear;
  • Select "Network";
  • Select "Advanced Settings";
  • Scroll to "DNS Settings" and select "Manual";
  • Put "107.170.15.247" as primary DNS and "77.88.8.8" as secondary;
  • Save changes and Xbox One will now check the internet connection.

    Setup DNS address

  • Press "Menu" button on remote, scroll down to "Network" and chose "Network Settings";
  • Select type of your internet connection and then select "IP Settings";
  • Scroll down to DNS settings and put "107.170.15.247" as primary DNS and "77.88.8.8" as secondary;
  • Your connection will test itself. Press "Next" to save the settings and exit the menu.

    • iPhone

  • Launch "Settings" and click on Wi-Fi;

  • Click on the DNS field;
  • Fill DNS separated by comma: "107.170.15.247, 77.88.8.8".

    DNSmasq

    If you use local DNS server, like DNSmasq, you can allow to enable custom DNS settings on particular domains. Add host that you want unlock to dnsmasq configuration file and restart service:

    $ nano /etc/dnsmasq.conf

    server=/netflix.com/107.170.15.247
    server=/pandora.com/107.170.15.247
    ...

    $ service dnsmasq restart
