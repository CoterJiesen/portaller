 How to set up Portaller in Windows

1. Open «Network center»

2. Select active connection

3. Click «Options» in the new window

4. Select IPv4 row and choose «Options»

5. Set Portaller IP shown above in «Preferred DNS» section

 How to set up Portaller in Mac OS

1. Open «Network» section

2. Select active connection and click «Options»

3. Open «DNS» section

4. Click «Add» and set Portaller IP shown above
